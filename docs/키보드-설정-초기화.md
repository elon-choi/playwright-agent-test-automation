# 맥 키보드 설정 초기화 (외장 키보드 제거 후)

외장 키보드가 없어졌을 때, 해당 키보드로 저장됐던 설정을 지우려면 아래 순서대로 진행하면 됩니다.

---

## 1. 외장 키보드 타입 정보 삭제 (권장)

시스템에 저장된 **키보드 타입/매핑 정보**를 지우면, 다음에 키보드를 연결했을 때 다시 “키보드 식별” 단계가 뜹니다. 지금은 외장 키보드를 쓰지 않으므로, 이 파일만 지워도 됩니다.

**터미널에서 실행 (비밀번호 입력 필요):**

```bash
sudo rm -f /Library/Preferences/com.apple.keyboardtype.plist
```

실행 후 **재부팅** 한 번 하면 적용됩니다.

---

## 2. 시스템 설정에서 확인 (선택)

- **시스템 설정 → 키보드**
  - **키보드 단축키** → **입력 소스**  
    - “이전 입력 소스 선택”이 원하는 대로인지 확인 (보통 Ctrl+Space).
  - 하단 **“기본값으로 복원”** 이 있으면, 키보드 관련만 초기화하고 싶을 때 사용할 수 있습니다.  
    (전체 키보드 단축키가 기본값으로 돌아가므로, 필요한 단축키는 다시 설정해야 할 수 있습니다.)

---

## 3. 사용자 키보드 관련 설정 (필요할 때만)

다른 맥과 비교했을 때 **입력 소스 단축키는 동일**하다고 하셨으므로, 보통은 **1번만** 하면 됩니다.

그래도 한/영 키 동작이 이상하면:

- **시스템 설정 → 키보드 → 입력 소스**  
  - “이전 입력 소스 선택” / “입력 메뉴에서 다음 소스 선택” 단축키를 **기본값**으로 되돌리거나,  
  - 한/영에 쓰는 단축키를 **다른 키**(예: F18)로 바꿔 보는 방법도 있습니다.

---

## 요약

1. 터미널에서 `sudo rm -f /Library/Preferences/com.apple.keyboardtype.plist` 실행  
2. 재부팅  
3. 필요하면 시스템 설정 → 키보드에서 “기본값으로 복원” 또는 입력 소스 단축키 확인  

이렇게 하면 “윈도우 외장 키보드” 때문에 남아 있던 키보드 설정은 초기화된 상태에 가깝게 정리됩니다.
