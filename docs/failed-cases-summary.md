## 실제 시나리오 실패 확인 방법

로컬 터미널에서 아래 명령으로 재실행 후, HTML 리포트에서 실패한 케이스만 확인할 수 있습니다.

```bash
cd /Volumes/Project/playwright-agent-test-automation
CI=true npx playwright test --project=chromium --project=chromium-mw
npx playwright show-report
```

또는 JSON 결과에서 실패만 추출:

```bash
node -e "
const r = require('./test-results/results.json');
r.suites?.forEach(s => s.specs?.forEach(sp => sp.tests?.forEach(t => {
  if (t.results?.some(x => x.status === 'failed')) console.log(sp.title, '->', t.results.find(x => x.status === 'failed')?.error?.message?.split('\n')[0]);
})));
"
```
