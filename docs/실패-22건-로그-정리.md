# 전체 시나리오 실패 로그 정리 (90 통과 / 22 실패)

`test-results/results.json` 기준으로 추출한 실패 목록입니다. (동일 run에서 실패가 14건으로 수집된 경우, 나머지 8건은 다른 프로젝트/필터 run일 수 있습니다.)

## 1. KPA-048 · 운영 중인 배너의 UI 및 동작 검증
- **에러:** 배너 이동 URL이 일치하지 않습니다. 기대: `https://page.kakao.com/open/webview/event?hash_uid=42e205873ca6343edae005e60af9c6be` / 현재: `https://page.kakao.com/`
- **원인 추정:** 배너 클릭 후 이벤트 URL로 이동하지 않고 메인(/)으로 남음. 팝업/새 탭 처리 또는 네비게이션 검증 로직 이슈.

---

## 2. KPA-051 · 운영중인 배너의 동작 및 노출 검증
- **에러:** `cdpSession.send: Protocol error (DOM.getContentQuads): No node found for given backend id`
- **원인 추정:** ZeroStep AI fallback 사용 중 DOM 노드가 이미 제거됨. 배너 화살표 클릭 시 요소 비유지/타이밍 이슈.

---

## 3. KPA-054 · 운영중인 배너의 상호작용 및 이동 검증
- **에러:** `cdpSession.send: Protocol error (DOM.scrollIntoViewIfNeeded): Node is detached from document`
- **원인 추정:** ZeroStep 사용 시 노드가 document에서 제거된 상태. 배너 영역 스크롤/클릭 시 타이밍·재조회 필요.

---

## 4. KPA-057 · 운영중인 배너의 동작 및 노출 검증
- **에러:** 배너 이동 URL이 일치하지 않습니다. 기대: `https://page.kakao.com/open/webview/event?hash_uid=e4ede14aa4d97cb10f3a6918ca24ede6` / 현재: `https://page.kakao.com/menu/10016`
- **원인 추정:** 배너 클릭 후 이벤트 URL 대신 메뉴 페이지(/menu/10016)로 이동. 클릭 대상 또는 네비게이션 처리 이슈.

---

## 5. KPA-062 · 이벤트 페이지 UI 및 흐름 검증
- **에러:** `locator.click: Element is outside of the viewport`
- **원인 추정:** "이벤트 전체 보기" 배너가 뷰포트 밖에 있음. scrollIntoViewIfNeeded 후에도 클릭 실패. 스크롤·대기 보강 필요.

---

## 6. KPA-008 · 비로그인 상태에서 프로필 아이콘 클릭 시 로그인 페이지로 이동
- **에러:** 프로필 아이콘으로 클릭한 요소가 로그인 페이지로 이동시켰습니다. 00-login.feature 실행 후 다시 시도하세요.
- **원인 추정:** KPA-008은 **비로그인** 시나리오인데, storage state로 로그인된 상태에서 실행되어 프로필 클릭 시 로그인 페이지로 가지 않음 → 검증 실패. 또는 실행 순서/컨텍스트가 로그인 상태로 되어 있음.

---

## 7. KPA-105 · 댓글 기능 검증
- **에러:** `expect(received).toBe(expected) // Object.is equality`
- **원인 추정:** 댓글 관련 어설션(개수/내용 등) 불일치. 구체적 expect 조건 확인 필요.

---

## 8. KPA-109 · 무료 회차 진입 및 뒤로 가기 기능 검증
- **에러:** `TimeoutError: locator.waitFor: Timeout 10000ms exceeded.`
- **원인 추정:** 무료 회차 진입 또는 뒤로가기 후 대기하는 locator가 10초 내에 나타나지 않음. 선택자/페이지 구조/로딩 시간 확인.

---

## 9. KPA-113 · 무료 회차 댓글창 팝업 및 종료 검증
- **에러:** `TimeoutError: locator.waitFor: Timeout 10000ms exceeded.`
- **원인 추정:** 댓글창 팝업 또는 종료 후 기대 요소가 10초 내에 보이지 않음.

---

## 10. KPA-116 · 무료 회차에서 원작소설로 이동
- **에러:** 원작소설로 보기 배너를 찾지 못했습니다. 최하단까지 스크롤 후에도 노출되지 않습니다.
- **원인 추정:** 해당 작품/회차에 "원작소설로 보기" 배너가 없거나, 선택자/스크롤 로직이 실제 UI와 불일치.

---

## 11. KPA-127 · 무료 회차에서 정주행 기능 검증
- **에러:** 정주행 시 '다음화 무료로 보기' 메뉴가 미노출되어야 함
- **원인 추정:** 정주행 상태에서 "다음화 무료로 보기"가 보임(기대는 미노출). UI 변경 또는 조건(회차/이용권) 불일치.

---

## 12. KPA-044-1 · 삭제 확인 팝업에서 취소 시 구매작품 리스트 유지
- **에러:** `TimeoutError: locator.click: Timeout 15000ms exceeded.`
- **원인 추정:** 삭제 확인 팝업에서 취소 버튼 클릭 시 15초 내 요소 미노출 또는 클릭 불가. 팝업 구조/선택자 확인.

---

## 13. KPA-093 · 회차 리스트와 뷰어 이동 검증
- **에러:** `expect(page).toHaveURL(expected) failed`
- **원인 추정:** 회차 클릭 후 기대 URL로 이동하지 않음. URL 패턴 또는 리다이렉트/탭 처리 이슈.

## 14. KPA-002 · 사용자가 카카오 페이지에 로그인하여 추천홈에 진입
